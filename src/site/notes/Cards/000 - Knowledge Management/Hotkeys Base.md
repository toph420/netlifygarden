# Hotkeys Base

`cmd + P` open command pallet
`cmd + D` toggle source view
`cmd + E` toggle reading view
`cmd + shift + I` reveal in navigation
`cmd + 1` line up
`cmd + 2` line down
`cmd + 3` copy line up
`cmd + 4` copy line down
`option + M` move current file to folder
`option + A` back 
`option + S` forward
`option + D` datepicker
