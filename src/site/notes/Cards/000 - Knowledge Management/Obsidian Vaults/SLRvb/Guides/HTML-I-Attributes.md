# `<i></i>` Attributes

| Alt="\_" | Description |
| --- | --- |
| `left` | Move text of `<i>` to the left |
| `right` | Move text of `<i>` to the right |
| `no-bg` | Remove the background color of `<i>` |
| `clear` | Equivalent of a `<br>` or an enter. |
